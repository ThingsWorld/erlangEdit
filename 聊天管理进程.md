聊天管理进程：

chat_mgr.erl

```erlang
gen_server模块
1.增加数据到缓存中，使用ETS表
2.离线信息处理
3.增加角色离线信息
4.删除角色离线信息
5.聊天记录
6.推送聊天信息
7.加载数据
```



聊天处理

chat.erl

```erlang
1.角色登录信息聊天
2.翻译聊天结果然后广播
3.聊天语音
4.判断是否可以聊天：是否被禁言，公会等级是否满级，聊天的类型是否符合。
```

聊天监控

chat_monitor.erl

```erlang
gen_server模块
1.私聊聊天监控
2.加载数据
3.判断聊天信息
```




语音缓存

chat_voice_cache.erl

```erlang
1.gen_server模板
2.增加语音到缓存

4.增加语音处理
```



聊天远程过程调用协议（RPC处理）

chat_rpc.erl

这里写聊天相关的协议。

```erlang
1.私聊协议
2.角色离线协议
3.接收语音信息协议
4.请求语音缓存信息协议

```
